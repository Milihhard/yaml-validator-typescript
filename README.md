yaml-validator-typescript is a way to validate your Yaml file by using a defined model

## Installation

```bash
$npm install yaml-validator-typescript
```

or

```bash
$yarn add yaml-validator-typescript
```

### As a command

yaml-validator file.yaml model.ts
